# Pizza Sales Analysis Project

This project contains a SQL-based analysis of pizza sales data using CSV files. It is designed to demonstrate data cleaning, exploration, and business-focused SQL queries for a pizza restaurant dataset.

## Project Overview

The dataset includes information about:
- Orders
- Order details
- Pizza inventory and pricing
- Pizza types and categories

The analysis focuses on answering key business questions such as:
- Best-selling pizzas
- Revenue contribution by pizza type
- Order trends over time
- Popular pizza categories and sizes

## Dataset Files

The project includes the following CSV files:
- orders.csv
- order_details.csv
- pizzas.csv
- pizza_types.csv

## Tools Used

- SQL
- MySQL / PostgreSQL-compatible SQL queries
- CSV dataset files
- Git and GitHub for version control

## Project Structure

```text
pizza_sales_project/
├── orders.csv
├── order_details.csv
├── pizzas.csv
├── pizza_types.csv
```

## Example Analysis Questions

Some example questions covered in this project include:
1. Which pizza is the most ordered?
2. Which pizza contributes the highest revenue?
3. What are the most popular pizza sizes?
4. Which days or months show the highest sales?

## How to Use

1. Clone the repository.
2. Import the CSV files into your SQL database.
3. Run the SQL queries from your analysis workflow.
4. Explore the results and adapt them for reporting.
